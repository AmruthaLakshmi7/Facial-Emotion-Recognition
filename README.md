# Facial Emotion Recognition System

A machine learning / deep learning project that detects human emotions from facial images using computer vision and real-time webcam input.

## Features
- Detects emotions such as Happy, Sad, Angry, Neutral, etc.
- Real-time emotion detection using webcam
- Image preprocessing and face detection
- Trained deep learning model for classification

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy
- Matplotlib

## Project Structure
- train.py — Model training script
- test.py — Testing script
- main.py — Main execution file
- webcam.py — Real-time emotion detection using webcam
- emotion_model.h5 — Trained model file
